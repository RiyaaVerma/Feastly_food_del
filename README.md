# 🍔 Food Delivery App (MERN Stack)

A full-stack Food Delivery web application built using the MERN stack. This project allows users to browse food items, place orders, and make secure payments, while admins can manage products and orders efficiently.

---

## 🚀 Live Demo

* 🌐 Frontend: https://feastly-frontend1.onrender.com
* 🔗 Backend API: https://feastly-backend-i4v5.onrender.com
* 🗄️ Admin:https://feastly-admin.onrender.com

---

## 📌 Features

### 👤 User Features

* User authentication (Signup/Login)
* Browse food items
* Add items to cart
* Place orders
* Secure online payment integration (Stripe)
* View order history

### 🛠️ Admin Features

* Add / Update / Delete food items
* Manage orders (Accept / Reject)
* Dashboard with order status

---

## 🧰 Tech Stack

### Frontend

* React.js
* CSS / Modular Styling
* Axios

### Backend

* Node.js
* Express.js

### Database

* MongoDB (Mongoose)

### Payment Integration

* Stripe API

```

### 2️⃣ Setup Backend

```

```

---

### 3️⃣ Setup Frontend

```
cd frontend
